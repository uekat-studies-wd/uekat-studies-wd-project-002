# uekat-studies-wd-project-002

Uniwersytet Ekonimiczny in Katowice - project no. 2 for the subject "Wizualizacja danych"

## Local run

### Via docker

```bash
docker compose up -d
```

URL: http://localhost:5173

### Via node

```bash
yarn install
yarn dev
```

URL: http://localhost:5173

## Build

```bash
yarn build
```
